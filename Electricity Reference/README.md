# Reference calculation of electricity use in dissassembly
The disassembly of 1 m² gypsum board is used as a reference for the elextricity use in dissassembly.

## Assumptions
- There are 20 screws per m² gypsum board
- It takes 5 seconds to unscrew each screw using a 90 W cordless drill
- Gypsum board weighs 9.10 kg/m²

## Calculation
Based on the above mentioned assumptions:
- Unscrewing 1 screw requires 450 J
- Unscrewing 1 m² of gypsum board (20 screws) requires 9.00 kJ
- **The dissassembly of 1 kg material requires 989 J**.
